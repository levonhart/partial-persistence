## Build
Definir o compilador no arquivo Makefile definindo a variável CC (`CC=clang` por exemplo).

```
$ make
```

## Usage

```
$ ./main INPUT [OUTPUT]
```

INPUT é o arquivo de entrada, e OUTPUT, o arquivo de saída. Exemplo:

```
$ ./main random_data.txt output.txt
```
